# Document AI Invoice Processing Pipeline (Open-Source)

This project implements an end-to-end document processing pipeline that extracts structured data from unstructured PDF invoices using OCR and NLP techniques. The pipeline mimics enterprise Document AI systems while remaining fully open-source and free.

## Architecture

PDF Invoices → OCR → Field Extraction → Validation → Analytics Database

## Extracted Fields
- Invoice number
- Invoice date
- Supplier name
- Total amount
- Currency
- VAT amount

## Tech Stack
- Python
- Tesseract OCR
- SQLite
- Pandas

## Use Case
Automating invoice processing for financial analytics and reporting.
